Lightweight compare to pry or irb

Less lines of code, and starts faster

rib irb pry
lines of code 1500 4500 9000
launch time (ms) 377.5 867.2 891.3

Modular, extensible, and lots of plugins

Customization

Here is a selection of plugins:
  • completion
  • history
  • multiline
  • readline
  • strip backtrace

Less complexity and less bugs

rib uses Ruby's internal lexical parser, while irb implements a separate parser.
This make rib behaves more faithful to Ruby and has less bugs.
For example, in irb, you can't use `:if` symbol key in hash, but in rib you can.

gem install rib

Usage

After install, you can start the shell by calling: rib If you use Rails, then call: rib rails
